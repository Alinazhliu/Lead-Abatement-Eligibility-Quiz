# Lead Eligibility Quiz Widget
This is a JavaScript-based quiz to determine eligibility for Habitat for Humanity lead-safety programs.

## How to Use
1. Download the folder.
2. Open `index.html` in a browser to see the quiz in action.
3. To integrate into an existing website:
   - Copy the `<div id="app"></div>` into your HTML page.
   - Include `style.css` and `script.js`.
   - Make sure the paths are correct.
4. Supports English, Spanish, and Chinese via the language buttons.

## Customization
- Questions and blurbs are in `script.js` under the `content` object.
- Income limits are in the `limits` object.
